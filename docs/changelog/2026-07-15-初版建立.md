# 初版建立

- 日期:2026-07-15
- 版本:→ 1.0.0
- 類型:新功能

## 需求

建立一個 .NET 10 / C# Console CLI 工具,將指定來源目錄下的 Office、PDF、Markdown、
Text、HTML、CSV、JSON、XML 檔案批次轉換為純文字內容,輸出到指定目錄供知識庫使用;
不符合條件的檔案跳過。需讀取 `appsettings.json` 設定,並提供多等級日誌輸出(含例外)。

## 變更

### 專案結構

- 新增 `src/TextConvert`(Console 專案,net10.0)與 `src/TextConvert.Tests`(xUnit)。
- 新增 `.gitignore`(忽略 `bin/`、`obj/`、`logs/` 等)。

### 轉換器(`Converters/`)

- `IDocumentConverter` 介面。
- `WordConverter`、`ExcelConverter`、`PowerPointConverter`(OpenXML SDK)。
- `PdfConverter`(PdfPig,僅內嵌文字)。
- `HtmlConverter`(HtmlAgilityPack)、`MarkdownConverter`(Markdig)。
- `CsvConverter`(RFC 4180 解析,Tab 分隔)。
- `TextConverter`(`.txt`/`.text`/`.json`/`.xml` 原樣輸出)。

### 服務(`Services/`)

- `FileScanner`:遞迴 / 非遞迴列舉檔案。
- `ConverterRegistry`:依副檔名解析轉換器。
- `ConversionService`:掃描→轉換→鏡像寫出,單檔失敗不中斷並輸出統計。

### 設定與日誌

- `appsettings.json`:`Version`、`Conversion`(來源 / 輸出 / 遞迴 / 覆寫)、`Serilog`。
- `Program.cs`:載入設定、初始化 Serilog、命令列 `--source`/`--output` 覆寫。

### 版本

- `appsettings.json` 版本設定為 `1.0.0`。

## 範圍與已知限制

- 不支援舊版二進位 Office(`.doc`/`.xls`/`.ppt`)。
- 不支援影像型 PDF 的 OCR。
- 輸出僅純文字 `.txt`。

## 影響範圍

- 新增:`src/` 全部程式與測試、`docs/` 文件、根目錄 `.gitignore`、`README.md`。

## 驗證

- `dotnet build`:成功(0 警告 0 錯誤)。
- `dotnet test`:32 項測試全數通過。
- 端到端:對含 txt/md/html/csv/json 與不支援 `.zip` 的目錄執行,
