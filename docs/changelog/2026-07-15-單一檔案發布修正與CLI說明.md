# 單一檔案發布修正與 CLI 說明參數

- 日期:2026-07-15
- 版本:1.0.0 → 1.0.1
- 類型:修正 + 新功能 + 文件

## 需求

1. 修正發布版(自封裝單一檔案 win-x64)執行時的崩潰。
2. 新增命令列說明參數 `-h` / `--help`,以及版本參數 `--version`。
3. 撰寫針對發布版執行檔的使用者操作手冊,並隨發布輸出附帶。

## 變更

### 修正:單一檔案發布下 Serilog 崩潰

- 現象:發布為單一檔案後,執行任何指令(含 `-h`)都拋出
  `InvalidOperationException: No Serilog:Using configuration section is defined ...`,
  導致發布版完全不可用。
- 原因:單一檔案發布下 Serilog.Settings.Configuration 無法透過組件掃描找到 sink 組件。
- 修法:於 `appsettings.json` 的 `Serilog` 區塊新增
  `"Using": [ "Serilog.Sinks.Console", "Serilog.Sinks.File" ]`,明確指定 sink 組件。

### 修正:主控台繁體中文亂碼

- 現象:發布版在 Windows 主控台輸出的繁體中文(help、版本、Serilog 主控台訊息)顯示為亂碼;
  檔案日誌則正常。
- 原因:`Console` 預設輸出編碼非 UTF-8。
- 修法:`Program.cs` 啟動時設定 `Console.OutputEncoding = Encoding.UTF8`。

### 新功能:CLI 說明與版本參數

- `CommandLineParser` 回傳型別擴充為 `CommandLineOptions`(新增 `ShowHelp`、`ShowVersion`)。
- 新增 `-h` / `--help` / `-?`(顯示說明)與 `--version` / `-v`(顯示版本)。
- 新增 `Cli/UsageText.cs` 提供繁中說明文字。
- `Program.cs`:help 於載入設定與初始化 Serilog **之前**優先處理;version 於載入設定後、
  初始化 Serilog 前處理。

### 文件

- 新增 `docs/guides/使用者操作說明.md` 與 `docs/guides/README.md`。
- 透過 `TextConvert.csproj` 連結,將手冊自動複製到建置/發布輸出目錄(單一來源)。
- 更新 `docs/README.md`(新增 guides 分類)、`docs/prd/文字轉換-prd.md`(參數與現況校正)、
  根 `README.md`(help/version 用法與發布說明)。
